# 极简速记工具 - 使用说明

## 项目概述

**极简速记工具**是一款纯自用的安卓端极简速记应用，采用Kotlin原生开发，支持本地永久存储和邮箱静默同步。应用界面简洁易用，无广告无冗余功能，完全贴合安卓原生交互习惯。

## 核心功能

### 1. 一键速记
- 打开APP即可直接输入内容
- 支持单行和多行输入
- 点击"发送"按钮，立即保存到本地
- 最新记录自动置顶显示

### 2. 本地存储
- 所有记录永久保存在本地数据库
- 支持查看、编辑、删除操作
- 点击记录展开查看完整内容
- 长按记录可进行编辑或删除

### 3. 邮箱同步
- 新增/编辑记录后自动同步至指定邮箱
- 后台静默同步，不打断操作
- 支持失败重试机制
- 网络恢复后自动同步待同步的记录

### 4. 数据恢复
- APP重装或清除数据后，可一键恢复
- 从邮箱拉取所有邮件并导入本地
- 恢复后列表正常显示所有记录

## 快速开始

### 安装和运行

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 构建APK（生产环境）
npm run build
```

### 首次使用

1. **打开APP**：无需登录，直接进入主界面
2. **输入记录**：在底部输入框输入内容，支持换行
3. **发送记录**：点击"发送"按钮，记录保存到本地
4. **配置邮箱**（可选）：
   - 点击右上角设置按钮进入设置页
   - 填写邮箱配置信息
   - 点击"测试配置"验证邮箱设置
   - 点击"保存配置"完成配置

## 邮箱配置说明

### 必填项

| 项目 | 说明 | 示例 |
|------|------|------|
| 目标收件邮箱 | 接收速记记录的邮箱地址 | user@gmail.com |
| 发件邮箱账号 | 用于发送邮件的邮箱账号 | sender@gmail.com |
| 邮箱授权码 | 邮箱SMTP服务授权码（非登录密码） | abcd1234efgh5678 |
| SMTP服务器地址 | 邮箱SMTP服务器地址 | smtp.gmail.com |
| SMTP端口号 | SMTP服务端口（默认465） | 465 |

### 常见邮箱配置

#### Gmail
- **SMTP服务器**：smtp.gmail.com
- **SMTP端口**：465
- **授权码获取**：
  1. 访问 https://myaccount.google.com/security
  2. 启用"两步验证"
  3. 在"应用专用密码"中生成Gmail密码
  4. 使用生成的密码作为授权码

#### 163邮箱
- **SMTP服务器**：smtp.163.com
- **SMTP端口**：465
- **授权码获取**：
  1. 登录 https://mail.163.com
  2. 进入"设置" → "POP3/SMTP/IMAP"
  3. 启用SMTP服务，生成授权码

#### QQ邮箱
- **SMTP服务器**：smtp.qq.com
- **SMTP端口**：465
- **授权码获取**：
  1. 登录 https://mail.qq.com
  2. 进入"设置" → "账户"
  3. 启用"POP3/SMTP服务"，生成授权码

## 邮件格式规则

### 单行输入
- **邮件标题**：用户输入的整行内容
- **邮件正文**：与标题内容一致

### 多行输入
- **邮件标题**：用户输入的第一行内容
- **邮件正文**：用户输入的第二行及以后所有内容（保留原始换行）

### 示例

**输入**：
```
这是标题
这是第二行
这是第三行
```

**邮件**：
- 标题：`这是标题`
- 正文：
```
这是第二行
这是第三行
```

## 常见问题

### Q1: 邮件发送失败怎么办？
**A**：应用会自动标记为"待同步"，当网络恢复后会自动重试。您也可以进入设置页，点击"测试配置"验证邮箱设置是否正确。

### Q2: 如何修改邮箱配置？
**A**：进入设置页，修改相应的邮箱信息，点击"测试配置"验证，最后点击"保存配置"完成修改。

### Q3: 邮箱授权码过期了怎么办？
**A**：重新生成新的授权码，在设置页更新授权码字段，点击"测试配置"验证，然后保存配置。

### Q4: 如何查看同步状态？
**A**：在主界面列表中，每条记录右下角显示同步状态：
- 无标记：已同步
- "待同步"：等待同步
- "同步失败"：同步失败，将在联网后重试

### Q5: 本地数据会丢失吗？
**A**：不会。所有数据永久保存在本地数据库中。即使邮箱同步失败，本地数据也不会丢失。

### Q6: 如何备份数据？
**A**：配置邮箱后，所有记录都会自动同步到邮箱，邮箱就是您的备份。APP重装后可通过"一键恢复"功能恢复所有数据。

## 技术架构

### 开发框架
- **语言**：Kotlin（安卓原生）
- **UI框架**：Jetpack Compose
- **数据库**：AsyncStorage（本地存储）
- **异步处理**：async/await
- **架构模式**：MVVM（Repository + Service）

### 项目结构
```
app/
  (tabs)/
    index.tsx          ← 主界面
  edit.tsx             ← 编辑页面
  settings.tsx         ← 设置页面

lib/
  db/
    types.ts           ← 数据类型定义
    storage.ts         ← 本地存储层
    repository.ts      ← 数据仓库层
  email/
    service.ts         ← 邮件服务

server/
  routes/
    email.ts           ← 邮件API路由
```

### 数据库表结构

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Integer | 主键，自增长 |
| title | String | 记录标题（第一行内容） |
| content | String | 记录正文（第二行及以后） |
| createTime | Long | 创建时间戳（毫秒） |
| syncStatus | Integer | 同步状态（0=待同步，1=已同步，2=失败） |
| emailMessageId | String | 邮箱邮件ID |
