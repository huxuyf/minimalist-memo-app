# 极简速记工具 - 界面设计规划

## 设计原则
- **极简风格**：白色背景+深灰色文字，无花哨设计
- **移动优先**：竖屏（9:16）、单手操作友好
- **安卓原生**：遵循Material Design 3规范，符合安卓交互习惯
- **高效操作**：一键速记，无冗余操作步骤

## 屏幕列表

### 1. 主界面（Home Screen）
**功能**：显示速记列表和输入区域

**布局结构**：
- 顶部：应用标题 + 设置按钮
- 中部：记录列表区（全屏展示，最新记录置顶）
- 底部：固定输入框 + 发送按钮

**内容展示**：
- 列表项：仅显示标题（第一行内容）
- 点击展开：显示完整正文
- 长按菜单：编辑、删除操作

**交互流程**：
1. 打开APP → 直接进入主界面，秒开加载本地记录
2. 输入内容 → 点击发送 → 本地保存 + 静默同步邮箱
3. 发送成功 → Toast提示"已保存并同步"
4. 发送失败 → Toast提示"同步失败，将在联网后重试"

### 2. 编辑对话框（Edit Dialog）
**功能**：编辑已有记录

**布局**：
- 标题栏：关闭按钮 + "编辑记录"标题 + 保存按钮
- 内容区：多行文本编辑框（预填充原内容）
- 底部：取消、保存按钮

**交互流程**：
1. 长按列表记录 → 选择"编辑" → 打开编辑对话框
2. 修改内容 → 点击保存 → 更新本地数据 + 重新同步邮箱
3. 点击取消 → 关闭对话框，不保存修改

### 3. 删除确认对话框（Delete Confirmation）
**功能**：二次确认删除操作

**布局**：
- 提示文本："确认删除此记录？"
- 按钮：取消、确认删除

**交互流程**：
1. 长按列表记录 → 选择"删除" → 弹出确认对话框
2. 点击确认 → 删除本地记录 + 可选同步删除邮箱邮件

### 4. 设置页面（Settings Screen）
**功能**：邮箱配置和应用设置

**布局结构**：
- 顶部：返回按钮 + "设置"标题
- 邮箱配置区：
  - 目标收件邮箱（输入框）
  - 发件邮箱账号（输入框）
  - 邮箱授权码（密码框，隐藏显示）
  - SMTP服务器地址（输入框，可选下拉选择）
  - SMTP端口号（输入框，默认465）
- 操作按钮：
  - 测试邮件（验证配置）
  - 保存配置
  - 一键恢复（批量导入邮箱数据）

**交互流程**：
1. 点击主界面设置按钮 → 进入设置页
2. 填写邮箱配置信息 → 点击测试邮件 → 验证配置有效性
3. 点击保存 → 加密存储配置信息
4. 点击一键恢复 → 拉取邮箱所有邮件 → 批量导入本地数据库

## 技术实现细节

### 数据库表结构
```sql
CREATE TABLE memos (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  title TEXT NOT NULL,           -- 第一行内容
  content TEXT,                  -- 第二行及以后内容
  createTime INTEGER NOT NULL,   -- 时间戳（毫秒）
  syncStatus INTEGER DEFAULT 0,  -- 0=待同步，1=已同步，2=同步失败
  emailMessageId TEXT            -- 邮箱邮件ID（用于更新/删除）
);
```

### 邮件格式规则
- **单行输入**：邮件标题=用户输入，邮件正文=用户输入
- **多行输入**：邮件标题=第一行，邮件正文=第二行及以后（保留换行）
- **无时间戳**：邮件标题和正文完全贴合用户输入

### 同步机制
1. **新增/编辑**：后台异步通过SMTP发送邮件
2. **失败重试**：指数退避策略，标记为"待同步"，联网后自动重试
3. **删除同步**：可选配置是否同步删除邮箱邮件（默认开启）

## 性能目标
- 冷启动：≤1秒
- 热启动：≤0.5秒
- 操作响应：即时响应，无延迟
- 本地存储：极小占用空间
- 后台同步：不消耗额外电量和流量
